About

The Layout Shift Challenge was created by Request Metrics to demonstrate the frustrating behavior of Layout shifts and the Cumulative Layout Shift (CLS) metric for web performance.

Layout shifts are performance events caused by lazy-loading content into a website that changes position of other elements. For example, an image loading and pushing down text below it.

Cumulative Layout Shift (CLS) is a Core Web Vital performance metric that measures the total size and distance of these layout shifts. Poor CLS scores, can penalize your pagerank and decrease your traffic.

You can check if your website has CLS problems using the free Speed Check Tool.
